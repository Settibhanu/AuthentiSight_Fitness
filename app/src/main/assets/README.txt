Place the following generated DLC files in this directory before building:

  Quant_yoloNas_s_320.dlc   - Generated by Generate_models/GenerateDLC.ipynb (Cell 3)
                               OR downloaded from https://aihub.qualcomm.com (YOLO-NAS-S, SM8750)

  hrnet_axis_int8.dlc        - Generated by Generate_models/GenerateDLC.ipynb (Cell 5)
                               OR downloaded from https://aihub.qualcomm.com (HRNet-W32, SM8750)

And place avatar GLB files in:
  avatars/male.glb            - glTF 2.0 humanoid rig (male)
  avatars/female.glb          - glTF 2.0 humanoid rig (female)

Free sources for GLB humanoid rigs:
  https://sketchfab.com (filter: glTF, free, humanoid)
  https://www.mixamo.com (export as FBX, convert to glTF via Blender)
