Place the following audio file in this directory:

  alert_beep.mp3   - Short beep sound for posture alerts (< 1 second, mono, 44.1kHz)

This file is bundled in the APK. Posture voice alerts are delivered via Android TTS
(AlertSoundPlayer) and do NOT require a bundled audio file. This beep is used as
a supplementary non-speech notification.
